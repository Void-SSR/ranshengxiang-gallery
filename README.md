# 苒盛香 · 祖传牛舌饼包装策略

这是用于客户预选包装设计方向的 30 天 GitHub Pages 测试版。

- A、B、C 类方案浏览
- 手机/电脑排版切换
- 隐藏文字，只看图片
- 本机预选与 2—6 图对比组
- 一键复制预选结果
- 列表使用高清预览图，点击放大加载原图
- 首次进入时缓存80张高清预览图，随后以低并发在后台缓存无损原图
- 缓存前检查设备可用空间与省流量设置，空间不足时自动降级为重点预览或按需原图
- 同一浏览器二次进入会校验首尾代表图，防止缓存已被系统清理但本地标记仍存在

客户选择保存在各自浏览器的 `localStorage` 中，不会在不同设备之间同步。

## 离线 PDF

运行 `python3 scripts/generate-offline-pdfs.py` 会直接读取 `assets/originals` 内的 80 张 PNG 原图，在项目级 `output/pdf/` 生成手机单列版和电脑六图对比版。

## 重新生成高清预览图

列表图从对应原图重新转换为原尺寸、JPEG 94 质量的 sRGB 图片，不做自动提亮或色彩增强：

```bash
node scripts/regenerate-previews.mjs
```
